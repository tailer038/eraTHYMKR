에라토호YM 패러미터 컬러바 표시 기능/옵션 추가 패치

패치 적용 대상 버전 : eraYm 국내 갱신판(140722)


변경시 조교 화면은 02.png 참고




※ 적용 방법

1. ERB 폴더를 덮어씌운다
2. 옵션에서 표시 형식을 변경한다. 이하의 2가지 방법 중 하나를 택일.
   1) 메인 화면에서 [777] - 컨피그 설정 → [0] SHOP·이벤트 확장 → [1] 표시형식 변경
   2) [100] 조교화면    에서 → 표시형식변경[876]

   3(컬러바 형태) 식으로 설정하면 패러미터가 컬러바 형태로 표시된다.

   ※ 패치 적용 후 제일 첫번째에는 2->3이 아니라 2->2로 한 번 헛바퀴 돌지만 이후로는 정상 작동.
