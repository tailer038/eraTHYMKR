*何がどーなるのか*
**工作技能の効能**
工作技能を持つキャラクターは以下の事が出来るようになります。
（ただし(未)は未実装）

== Lv1 ==
    ・工作技能の成長
    （調教後レベルアップでは工作技能をLv0からLv1にすることは出来ません）

== Lv2 ==
    ・単純構造の調教器具の上質化
    ・単純構造のSM用品の上質化

== Lv3 ==
    ・電動調教機具の高性能化
    ・電動SM用品の高性能化
    ・単純構造の調教器具の特製品化
    ・単純構造のSM用品の特製品化
    ・工房での他の工作技能を持たない奴隷への工作技能指導

== Lv4 ==
    ・電動調教器具の特製品化
    ・電動SM用品の特製品化
(未)・工房での単純構造の調教器具製作／販売
(未)・工房での単純構造のSM用品製作／販売

== Lv5 ==
(未)・工房での電動調教器具製作／販売
(未)・工房での電動SM用品製作／販売
(未)・工房で特殊調教器具の製作

**特製品のオリジナリティ**
詳しくは技師パッチ用口上テンプレート(.ERB)を参照。
@KOJO_SPECIALNAME_CUSTOMTOOL_KXXで"STR:2 = "の後にアイテム名を書くことで、
「○○特製××」というアイテムがその名前に置き換わります。
また、一部の地の文に表示されるアイテム名にも影響します。
なお、コンフィグにより無効化できます。

**改造アイテムの確認**
改造した／改造予定の調教アイテムについて、
その状況をステータス画面で見ることができます。

**アイテム改造にまつわる口上群**
詳しくは技師パッチ用口上テンプレート(.ERB)を参照。
　@KOJO_DEVELOP_CUSTOMTOOL_KXX
　　…上質／高性能化の改良を被調教時に思いついた時
　@KOJO_DEVELOP_CUSTOMTOOL_PLUS_KXX
　　…特製品化の改良を被調教時に思いついた時
　@KOJO_DEVELOP_CUSTOMTOOL_ASSI_KXX
　　…上質／高性能化の改良を助手時に思いついた時
　@KOJO_DEVELOP_CUSTOMTOOL_ASSI_PLUS_KXX
　　…特製品化の改良を助手時に思いついた時
　@KOJO_DEVELOPED_CUSTOMTOOL_KXX
　　…上質／高性能化の改良が終わって調教前に披露する時
　@KOJO_DEVELOPED_CUSTOMTOOL_PLUS_KXX
　　…特製品化の改良が終わって調教前に披露する時
以上のタイミングに口上を追加することができます。

**工房**
建てて入れるようになりました。
また、奴隷を呼べます。
工房内では奴隷達に以下の作業を行わせることが出来ます。
また、主人も同様の作業を行えます。
・待機
  工房で待機させます。待機中、暇している奴隷達は何かするかもしれません。
・工作技能教習
  工作技能がLv3以上の奴隷と工作技能が無い奴隷が工房にいる場合、
  工作技能を伝授させることができます。
  ただし、教習にはそれなりの時間が必要になります。
