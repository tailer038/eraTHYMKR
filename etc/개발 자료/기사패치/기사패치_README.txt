[eratohoYM 국내 통합판 용 기사 패치]
技師パッチYM ver 0.65(Mk-2 rev.5改3 用) (10/03/02更新)를 국내 통합판 기준으로 적용했습니다.
150731-2 버전 기준으로 작업하였습니다.

<설치 방법>
eratohoYM 본체가 있는 곳에 덮어쓰면 됩니다.

<조교 기구 개조>
SHOP 화면에서 [777] 컨피그 설정 -> [6] 공작 기능 패치 -> [0] 조교 기구 개조 개발 기능 사용을 활성화하면 동작합니다.
도구를 개발하는 데에 필요한 능력과 소질을 만족하고 조수 가능한 캐릭터가 조교 대상이나 조수일 때 개발 중이 아닌 도구를 사용하면 조교 기구 개조에 착수합니다.
    A) 개조되지 않은 도구
    a, b를 모두 만족하면 고성능 도구 개발 착수.
        a. SM용품이 아니면 자위 중독 Lv1 이상
        SM용품일 경우 다음 4패턴 중 1개 이상 만족
        - 기교 Lv2 이상, 새드끼 Lv3 이상
        - 기교 Lv2 이상, 새드계 소질([초S], [새드]) 보유
        - 순종 Lv3 이상, 마조끼 Lv3 이상, 자위 중독 Lv1 이상
        - 자위 중독 Lv1 이상, 마조계 소질([초M], [마조]) 보유
        b. 전기 용품일 경우 공작 기능 Lv3 이상, 전기 용품이 아닌 경우 공작 기능 Lv2 이상
    B) 고성능 도구
    a, b를 모두 만족하면 특제 도구 개발 착수.
        a. SM용품이 아닐 경우 다음 2패턴 중 1개 이상 만족
        - 자위 중독 Lv3 이상, 함락계 소질(연모, 망신, 친애, 상애, 복종, 낙인, 예속, 사역마) 또는 음란계 소질(음란, 음핵, 음호, 음고, 음유, 요도광, 키스마, 초S, 초M, 즉각함락) 보유
        - 자위 중독 Lv5 이상
        SM용품일 경우 다음 4패턴 중 1개 이상 만족
        - 기교 Lv3 이상, 새드끼 Lv3 이상, 기교+새드끼 합계 8 이상
        - 기교 Lv4 이상, 새드계 소질([초S], [새드]) 보유
        - 순종 Lv3 이상, 마조끼 Lv3 이상, 자위 중독 Lv3 이상이고 순종+마조끼 합계 8 이상
        - 자위 중독 Lv3 이상, 마조계 소질([초M], [마조]) 보유
        b. 전기 용품일 경우 공작 기능 Lv4 이상, 전기 용품이 아닌 경우 공작 기능 Lv3 이상

개발 착수한 캐릭터를 대상으로 조교를 시작할 때 개발이 완료됩니다. 하나씩 완료되며 특제품이 우선 순위가 높습니다.

<공방>
SHOP 화면에서 [777] 컨피그 설정 -> [6] 공작 기능 패치 -> [0] 조교 기구 개조 개발 기능 사용을 활성화 -> [4] 공방 사용을 활성화하면 사용 가능합니다.
공방의 개설은 1개 이상의 특제품을 만들었을 때 가능합니다.
공방에서는 공작 기능 Lv3 이상의 캐릭터가 교사가 되어 Lv0인 캐릭터에게 수업을 진행해서 Lv1로 올릴 수 있게 합니다. (이 패치를 적용하면 공작 기능 Lv0 상태에서는 습득의 구슬로 레벨 업을 할 수 없습니다.)
수업 중 수업 외의 이벤트가 발생할 수 있습니다.

<주의 사항>
국내 통합판에서 개별적으로 추가한 도구 (예: 요도바이브 등)의 경우 적용이 되지 않습니다. 
국내 통합판과 일판 YM Mk-2 rev.5改3의 환경이 다른 데다가 원본 패치 자체가 미완성이었기에 이를 적용했을 때 버그가 발생할 수도 있습니다.
플레이어의 성향에 따라서 공방의 에로 이벤트가 밸런스가 안 맞다고 여길 수도 있습니다.
이런 다양한 난점에 대해서는 양해 부탁드립니다.
